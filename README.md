# Livre Vivant - Marie de France

Interatieve website voor een opdracht Frans: een middeleeuwse tekst wordt als levende ervaring gepresenteerd met scenes, keuzes en een mini-quiz.

## Tech stack

- Vite (vanilla JavaScript)
- HTML/CSS/JS

## Features

- Verhaal in 3 scenes met klikbare interpretaties
- Keuzemoment met alternatieve uitkomst
- Mini-quiz over thema en begrip
- Responsive lay-out voor desktop en mobiel

## Scripts

- `npm run dev` start de ontwikkelserver
- `npm run build` maakt een productiebuild
- `npm run preview` bekijkt de productiebuild lokaal
